Основы

Привет! Здесь ты можешь узнать, как пользоваться моим ботом. Начнём с того, что все команды, распознаваемые ботом, работают за счёт ключевых слов, например, "Ютуб" или "Дефолт", которые обязательно должны содержаться в твоём сообщении.

Например, если ты хочешь найти картинку с зимними горами, то запрос может выглядеть так:

1)Пикча Горы Зима

2)пикча гОрЫ зИмА

3)Зима Горы пикча

Исключение из этого правила одно: чтобы получить погоду в одном городе по его названию или местоположению, не надо указывать ключевые слова, достаточно просто написать название города или скинуть координаты вложением(только мобильная версия).

К регистру бот не чувствителен, это на случай, если юзер предпочитает писать без шифта или использует ПК.

Некоторые команды работают с аргументами, некоторые - только в таком виде, который указан в руководстве. Ниже фукнции с возможностью добавить аргументы будут помечаться символом "*" возле описания, а сами аргументы будут описаны в таком стиле: <аргумент>

Для получения краткой справки в самом Телеграме нужно отправить боту команду /help либо /start.

Команды прогноза погоды

Как уже было сказано, для получения краткой справки по погоде в конкретном городе, достаточно просто написать его название или скинуть координаты. Кроме того, с недавнего времени бот создаёт учётную запись каждому пользователю, что ему хоть раз писал. Делается это ради того, чтобы каждый мог создать себе конфиг с избранными городами, с помощью команды "Дефолт".

Синтаксис:

>> Дефолт - выводит справки по всем городам, что были добавлены в избранное

>> Дефолт все - выводит список гоородов, которые были добавлены в избранное

>> Дефолт сброс - очищает список избранных городов

>> Дефолт город* <название города> - добавляет город в избранное, если его ещё нет в списке, или же удаляет, если он там есть

За прогноз отвечает открытое API OpenWeatherMap, за что ему огромное спасибо.

Команды для скачивания музыки

Самая тяжёлая категория команд для бота, позволяют найти любой трек или просто видео на YouTube длительностью до 15 минут и получить его в диалог в формате .mp3. Выполняется около 15-30 секунд, но работает стабильно, треки загружаются с обложкой, и, зачастую, если автор оформил название по шаблону "Исполнитель - Название трека" либо Исполнитель - это название канала, то и теги трека буду проставлены по, так сказать, ГОСТу).

Синтаксис:

>> Ютуб* <поисковый запрос> - ищет видео на YouTube и конвертирует его в .mp3, отсылая в диалог

>> Музяка - высылает в диалог случайную песню из сохранённых у меня на диске альбомов(только 320kbit/s)

Команды для поиска картинок

Эта категория выполняет поиск картинок в качестве 720р в базе PixaBay по тегам, указанным в запросе, и отсылает картинку в диалог. База довольно обширная и работает быстро, но, в силу её ориентированности на англоязычных пользователей, поэтому предпочтительно искать по тегам на английском.

Синтаксис:

>> Пикча* <поисковый запрос> - ищет картинку по по тегам из аргумента, разделяя их пробелами

>> Пикча - высылает случайную картинку с моего жёсткого диска

Команды генерации случайностей

Генерация случайного числа из диапазона, выбранного пользователем

Синтаксис:

>> Рандом - выдаёт случайное число от 1 до 100

>> Рандом* <число> - выдаёт случайное число от 0 до указанного

>> Рандом* <число> <число> - выдаёт случайное число в диапазоне от меньшего указанного до большего

Команды для анекдотов с озвучкой и без

Присылает случайный анекдот с сайта baneks.ru и, при желании, озвучивает его

Синтаксис:

>> Анек* - один случайный анекдот в текстовом формате

>> Анек войсом* - один анекдот в формате голосового сообщения

Обе команды выше можно дополнить через пробел цифрой, указывающая количество анекдотов, которые надо прислать за раз

Другие команды

Ламповые фишки бота и команды, не относящиеся к другим категориям

>> Спасибо - выразить благодарность боту и его создателю, отправьте как знак, что вам нравится бот. Я об этом сразу узнаю)

>> /команда - группа команд для администратора бота, их описание есть в коде на GitHub. Не являются особо секретными, но и перечислять их в данном руководстве особо смысла нет
